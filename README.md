# RestaurantInvoiceSystem-V2

An advanced console-based Restaurant POS (Point of Sale) system built with Java.

# Features
- Modular OOP architecture
- Add / remove items
- Discount & Tax calculation
- Payment validation with change calculation
- Structured receipt printing

# How to Run the Project
1. Clone the repository
2. Open project in Java IDE (NetBeans / IntelliJ)
3. Compile all .java files
4. Run Main.java
